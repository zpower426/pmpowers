# Execute Plan: Product Execution Command

Use this command to drive AI agents to execute PM tasks as planned.

## Intent
Automate heavy tasks such as research analysis and Spec drafting.

## Trigger Actions
1. Activate `subagent-driven-discovery` or `spec-driven-design`.
2. Automatically invoke `validation-guard` at each task node for self-checking.
3. Upon task completion, automatically update `artifact-provenance`.
