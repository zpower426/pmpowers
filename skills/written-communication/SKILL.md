# Written Communication: Professional Documentation and Asynchronous Collaboration

Writing is the lever of a PM's influence.

## Checklist

1. **Set Clear Structure** — Summary, Background, Core Content, Conclusion/Action Items.
2. **Apply the Pyramid Principle** — Lead with the conclusion, followed by logical support.
3. **Eliminate Vague Vocabulary** — Strictly avoid terms like "possible," "probably," or "best effort."
4. **Set Reading Expectations** — Clarify whether the document is "For Information," "Feedback Required," or "Approval Pending."

## Core Mandates

- **Conciseness**: If you can say it in 10 words, don't use 100.
- **Visualization**: Use lists, bold text, and charts to break up heavy paragraphs.
- **Searchability**: Ensure document titles and tags are easy to find in project management systems.
