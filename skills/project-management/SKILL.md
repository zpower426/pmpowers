# Project Management: Project Control and Risk Governance

The PM must be the last line of defense for project success.

## Checklist

1. **Establish Critical Path** — Identify which task delays will cause the entire project to slip.
2. **Develop Risk Register** — Identify risks, assess probability/impact, and define mitigation strategies.
3. **Manage Resource Dependencies** — Coordinate the delivery sequence between different modules and teams.
4. **Status Reporting & Transparency** — Synchronize project health (Red/Yellow/Green) with the entire team and senior management.

## Core Mandates

- **Anticipation Over Remediation**: Senior PMs can always spot signs of trouble before problems occur.
- **Scope Creep Management**: Strictly control requirement changes; any addition must be accompanied by a proportional exchange of resources or time.
- **Buffer Awareness**: Always reserve a 15-20% time buffer for unforeseen contingencies.
