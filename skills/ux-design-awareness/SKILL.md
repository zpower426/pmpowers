# UX Design Awareness: Interaction Logic and Experience Evaluation

A PM doesn't need to draw beautiful UIs, but must understand perfect interactions.

## Checklist

1. **Evaluate Information Architecture (IA)** — Can users find core functions within 3 clicks?
2. **Verify Interaction Consistency** — Do button styles, error messages, and feedback animations conform to global standards?
3. **Perform Heuristic Audit** — Apply Nielsen's ten principles: visibility of system status, match between system and real world, user control and freedom, consistency and standards, error prevention, etc.
4. **Identify Cognitive Load** — Is there too much distracting information on the page? Are core actions prominent?

## Core Mandates

- **Accessibility Thinking**: Consider usage scenarios for different devices, lighting conditions, and visually impaired users.
- **Friction Management**: Not all friction is bad (e.g., confirmation dialogs for high-risk operations).
- **Minimalism**: Every element added dilutes the value of existing elements.
