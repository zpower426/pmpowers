# Metrics Definition: Quantitative Metrics and Feedback Loops

What cannot be measured cannot be optimized.

## Checklist

1. **Define Core Events** — Which user actions reflect the delivery of core value?
2. **Develop Instrumentation Plan (Tracking)** — Event names, properties, and trigger conditions.
3. **Categorize Metric Attributes** —
   - **Outcome Metrics**: Retention, Revenue.
   - **Proxy Metrics (Behavioral)**: Search frequency, Share rate.
4. **Establish Dashboards** — Ensure real-time visibility of data.

## Core Mandates

- **Prevent Noise**: More tracking is not necessarily better. Every data point tracked must correspond to a specific decision.
- **Data Consistency**: Ensure that the logical meaning of backend tracking aligns with frontend tracking.
- **Interpretability**: Metrics must be understandable to team members without a data background.
