# Data Analysis: Quantitative Analysis and Experimental Evidence

Data doesn't lie, but it can mislead.

## Checklist

1. **Define Funnels** — What is the user conversion path? Where is the most severe churn?
2. **Execute Retention Cohort Analysis** — How does retention performance differ across various user cohorts over time?
3. **Perform Correlation Search** — Which core behavior is highly correlated with long-term user retention (The Magic Number)?
4. **Verify Experimental Conclusions (Post-Experiment Audit)** — Did the experimental results achieve statistical significance?

## Core Mandates

- **Reject Vanity Metrics**: Focus on "DAU/Day-1 Retention" rather than "Total Registered Users."
- **Segment and Sub-segment**: Global data often hides the truth. Perform segment analysis by channel, device, and region.
- **Correlation is Not Causation**: Seeing two metrics rise simultaneously does not mean A caused B.

## Expert Principles

Before looking at data, write down your **expected conclusion**. If the data does not match the expectation, that is where the real learning opportunity begins.
